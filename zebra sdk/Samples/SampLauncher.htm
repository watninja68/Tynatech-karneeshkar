RFID Developer Kit for C

The samples that begin with "Basic" are designed to be simplistic in nature, allowing the source code to be easily understood so that they can be used as a learning tool for those new to developing applications. The rest (Standard Samples) offer a more complex demonstration of how to interface with the API functions. Those new to programming are encouraged to start with the basic sample applications.

RFID3
BasicRFIDHost1 Shows how to use RFID3 APIs on a host PC to perform simple inventory, Read, Write, Lock and Kill remotely on an LLRP enabled FX Series RFID Readers. Host (PC) based application for FX Series RFID readers FX7500, FX9600
RFIDHostSample1 Demonstrates the use of RFID3 APIs on a host PC to perform advanced inventory, Read, Write, Lock and Kill on an LLRP enabled device such as XR400.  It also demonstrates filter usage, reader configuration and reader management.
